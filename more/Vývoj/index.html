    Vývoj FaceAPP

    Pokud se chcete podílet na dalším vývoji tohoto projektu, musíte nejdříve splnit všechny předpoklady a instalační kroky potřebné pro spuštění aplikace.

    Dále je doporučeno se obeznámit s použitými knihovnami, jazyky a algoritmy:
    - Vývoj androidových aplikací
    - Flutter
    - Google ML Kit
    - OpenCV
    - Dart
    - Kotlin

    Za zmínku stojí zmínit, že framework Flutter umožňuje vývoj aplikací pro různé platformy (Android - iOS - Windows - Linux). Momentálně je aplikace funkční pouze pro Android.
